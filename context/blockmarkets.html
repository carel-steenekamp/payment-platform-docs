Regulated User Network

Validators are operated by Regulated FSPs

OpenRUN is a fully EVM compatible Public Permissioned Network operated exclusively by regulated FSPs as validators.  It is a regulated shared ledger for FSPs that enable regulated user functions.

Wallets are KYCd with Regulated FSPs

OpenRUN’s Regulated User Wallets are all KYCd by regulated FSPs and each wallet directly links to a Key Individual at a regulated FSP through non-transferable Smart KYC NFTs that define how the wallet transacts.

Tokens are issued by Regulated FSPs

OpenRUN’s Regulated Tokens are permissioned in either native digital asset (ERC20 compatible) or fractionalised and fungible real-world-asset (ERC1400 compatible) form that require smart KYC NFTs to transact with.
